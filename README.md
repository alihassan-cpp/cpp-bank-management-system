# C++ Bank Management System

## Description
This project is a console-based Bank Management System developed in C++. It demonstrates Object-Oriented Programming (OOP) concepts such as classes, encapsulation, and the use of STL vectors.

The system allows users to create bank accounts, manage account information, and perform basic banking operations.

## Features
- Create bank account
- Display all accounts
- Deposit money
- Withdraw money (coming soon)
- Search account (coming soon)
- Delete account (coming soon)

## Concepts Used
- Object-Oriented Programming
- Classes and Objects
- Encapsulation
- STL Vector
- Menu-driven programming
